Design a modern mobile fintech app prototype for a student-focused micro-investing platform. The brand should feel youthful, minimal, trustworthy, and simple. Use a clean interface with rounded cards, soft shadows, consistent spacing, and a clear visual hierarchy. Avoid clutter. Leave a placeholder space for the logo at the top of key screens but do not design a logo.

Use a consistent visual theme across all screens. Primary color should be deep blue or indigo. Use mint green as an accent color for growth indicators and positive returns. Background should be soft off-white or very light grey. Use green to indicate gains and red for losses. Apply rounded corners between 16–20px. Use a modern, clean sans-serif font. Icons should be minimal and simple. The overall tone should be friendly yet professional. Subtle gradients can be used in hero sections.

Create the following mobile screens designed for iPhone 14 size:

Landing / Welcome Screen
Include a logo placeholder at the top. Add a strong headline: “Invest from £1. Start Small. Grow Smart.” Include a short supporting tagline underneath. Add a minimal illustration representing growth such as a plant or upward graph. Include a primary call-to-action button labeled “Get Started” and a secondary text button labeled “Log In.”

Sign Up / Create Account Screen
Design a clean and minimal form with fields for Name, Email, and Password. Include an optional toggle for student verification and a checkbox for agreeing to terms and conditions. Add a primary button labeled “Create Account.” Maintain generous spacing and a simple layout.

Login Screen
Include Email and Password fields, a “Forgot Password?” text link, and a primary button labeled “Log In.” Keep the layout consistent with the sign-up screen.

Onboarding Screens (2–3 screens)
First screen should communicate “Invest from as little as £1” with a simple supporting illustration.
Second screen should highlight built-in learning tools with an educational-themed icon.
Third screen should communicate “Low fees. No confusion.” Include a primary button labeled “Start Investing.” Maintain consistent styling across onboarding screens.

Dashboard (Main Screen)
Include a greeting such as “Good Afternoon, [Name].” Display total portfolio value prominently at the top with a daily gain or loss indicator. Add a line chart showing growth over time. Include quick action buttons such as “Add Money,” “Invest,” and “Learn.” Add a card for “Round-Up Investing” with a toggle. Include a bottom navigation bar with tabs: Home, Invest, Learn, Profile.

Invest Screen
Display a list of simplified portfolio options such as “Starter Portfolio,” “Sustainable Growth,” and “Tech Focus.” Show risk level indicators (Low, Medium, High) and estimated annual return percentages. Clearly display a minimum investment amount of £1. Include a primary button labeled “Invest Now.”

Learn Section
Design a card-based layout with topics such as “What is Investing?”, “Understanding Risk,” and “How Diversification Works.” Include a simple progress indicator for completed lessons. Maintain the same visual style as other screens.

Profile Page
Include account settings, linked bank information, student verification badge, notification settings, and Help & Support. Maintain consistent spacing, typography, and visual language.

Ensure the design system is consistent across all screens, with uniform button styles, color usage, spacing, and typography. Prioritize usability, clarity, and simplicity. The overall aesthetic should feel accessible to students while maintaining trustworthiness expected from a financial platform.
